#### 三个版本

* JavaSE：标准版，基于桌面应用。
* JavaEE：企业版，服务器端应用。
* JavaME：微型版，电子产品应用。

#### 特性

* 跨平台/可移植性
* 安全性
* 面向对象
* 简单性
* 高性能
* 分布式
* 多线程
* 健壮性

#### JVM、JRE和JDK

JVM（Java Virtual Machine）就是一个虚拟的用于执行bytecode字节码的”虚拟计算机”。

JRE（Java Runtime Environment ） 包含：Java虚拟机、库函数、运行Java应用程序所必须的文件。

JDK（Java  Development Kit）包含：包含JRE，以及增加编译器和调试器等用于程序开发的文件。
